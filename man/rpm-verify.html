This is a quick breakdown of output if using rpm to verify packages.

If there is NO output everthing apears to be OK (mind you that we are using a tool on a possibly compromised system if you want to make sure use a dead system with a clean rpm package. The following are breakdowns of what the output means.

The following are very worrisome and deserve immediate attention:

  • S - File has changed size
  • M - File mode has changed (permissions)
  • 5 - File md5 checksum has changed (have you modified it such as sudoers? Most deserving attention)
  • U - File owner has changed
  • G - File group has changed
The following are less worrisome but deserve some thought and attention:
  • D - Device major/minor number mismatch
  • L - readLink path mismatch
  • T - mTime differs


For more detailed information: man rpm
